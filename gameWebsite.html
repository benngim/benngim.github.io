About the Project

During my free time, I decided to create this project to help revise and further develop the skills I learnt during university. It also would serve as a way for me to learn and familiarise myself with Flask. Using Render, I also deployed the web app to allow people to view a demo of it in action. Though since this was deployed using the free tier on Render, the web app will have noticeable delay when first loading it.

The web app allows users to play three different Javascript games (Pong, Snake and Tic-Tac-Toe). Each game comes with different game modes and customisation options. For example, in Snake there are options to play with static enemies and/or toggle on special foods that give extra points. For Pong and Tic-Tac-Toe, single player and multiplayer modes are available. Options for sound and color themes were also implemented in each game.

A MySQL database was also utilised to implement a user login system and store information about the users and the games. This includes high scores for games and the last time a user played a game for example. Simple and complex SQL queries were then used to display various different information on the home page and user profile page.

Tech Used

Python, Flask, HTML, CSS, JavaScript, MySQL, Git
