Personal Project - Manga/Anime Discord Bot

About the Project

As a fan of manga and anime, I wanted to build something that would help keep me up to date when new chapters/episodes would come out. This discord bot was then created as a result of this. Users can add and delete titles to a manga/anime list using discord bot commands. This list is stored using a MongoDB database. The bot scans new posts from the anime and manga subreddits and then posts it to the specified discord channel if the title is in the list.

Tech Used

Python, MongoDB, Git
